# Fili Attire Set (Bard)

Empyrean reforged. Set bonus : **Augments songs**.

Redirections : `Fili Attire Set +1 / +2 / +3`.
Ne pas confondre avec : Brioso (BRD, artifact reforged), Bihu (BRD, relic reforged), Aoidos' (BRD, empyrean tier antérieur).

## Set bonus — Augments songs

Les chants confèrent aux destinataires un **status boost** en plus de leur effet normal. Le stat boosté dépend de l'**élément du chant**.

| Élément | Stat |
|---|---|
| Fire | STR |
| Earth | VIT |
| Water | MND |
| Wind | AGI |
| Ice | INT |
| Thunder | DEX |
| Light | CHR |
| Dark | **MP** |

| Pièces | Bonus |
|---|---|
| 1 | aucun |
| 2 | +1 stat / +10 MP |
| 3 | +2 stat / +20 MP |
| 4 | +3 stat / +30 MP |
| 5 | **+5 stat / +50 MP** |

Scaling non linéaire : le passage 4 → 5 pièces vaut +2 (vs +1 pour chaque palier précédent). Le full set est le seul palier rentable en marge.

### Activation

Dès **2 pièces ou plus** parmi :

- Empyrean Armor +2 (Aoidos' +2)
- Reforged Empyrean Armor (Fili)
- Reforged Empyrean Armor +1 / +2 / +3

**Mix & match libre entre tiers.**

> Structure d'activation identique à celle du set RDM Lethargy (`lethargy-armor-set.md`), mais l'effet est tout autre : ici pas de durée, un stat boost sur la cible.
