# Lethargy Armor Set (Red Mage)

Empyrean reforged. Set bonus : **Augments "Composure"**.

Redirections : `Lethargy Armor Set +1 / +2 / +3`.
Ne pas confondre avec : Atrophy (BLM), Vitiation (DRK), Estoqueur's (RDM, tier antérieur).

## Set bonus — Augments "Composure"

Augmente la durée de l'**Enhancing Magic** et de l'**Enfeebling Magic**.

> Les enhancing lancés **sur soi-même** ne sont **pas** affectés. (Le self-cast relève du ×3 / cap 30 min de l'ability Composure elle-même.)

| Pièces | Bonus durée |
|---|---|
| 2 | +10 % |
| 3 | +20 % |
| 4 | +35 % |
| 5 | +50 % |

Évalué **au moment du cast**.

### Activation

Le bonus s'active dès **2 pièces ou plus** parmi :

- Empyrean Armor +2
- Reforged Empyrean Armor
- Reforged Empyrean Armor +1
- Reforged Empyrean Armor +2
- Reforged Empyrean Armor +3

**Mix & match libre entre tiers** — un +1 et un +3 comptent bien pour 2 pièces.

### Stacking

`Enhancing Magic Duration +x%` trouvé sur d'autres pièces est **multiplicatif** avec le total du set. Les **augments** d'items sont multiplicatifs séparément, appliqués après.

```
Durée = (Base + 6s×RDM G2 Merit + 3s×Relic Hands G2 augment + RDM Job Points + gear en secondes)
      × (Augments Composure)
      × (Duration % gear + Naturalist's Roll)
      × (Gear augments)
      × (Rune Fencer Gifts)
```
